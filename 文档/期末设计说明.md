# 期末设计说明

## 一、基本信息

| 项目 | 内容 |
|------|------|
| 作品名称 | 丝路热度数据观察 —— 近五年甘肃各地旅游客流、消费与客源地变迁 |
| 作品类型 | 数据新闻 H5 交互可视化 |
| 技术栈 | HTML5 + CSS3 + JavaScript + ECharts 5.5 + Python |

## 二、选题背景

丝绸之路（甘肃段）是中国最具国际影响力的文化旅游带之一，涵盖敦煌莫高窟、张掖丹霞、兰州黄河风情线等世界级旅游资源。2019-2024年五年间，该区域旅游经济先后经历了"疫情前巅峰 → 2020年断崖式下跌 → 2021-2023年持续反弹 → 2024年全面超越"的完整周期。

## 三、数据来源

| 数据维度 | 来源 | 时间跨度 |
|----------|------|----------|
| 游客接待量 | 甘肃省文旅厅/各市统计局 | 2019.01-2024.10 |
| 人均文旅消费 | 携程/美团公开数据 | 2019.01-2024.10 |
| 客源省份分布 | 手机信令+交通票务数据 | 2024年度 |
| 线上点评数据 | 美团/携程/小红书 | 2019.01-2024.10 |
| 节假日峰值 | 各市文旅局假期统计 | 2019-2024 |

## 四、可视化设计

作品采用长滚动H5单页叙事结构，共6个章节+封面+结语：

1. **热力图**：三城月度客流热度（城市可切换），暖色调渐变
2. **折线图+柱状图**：客流趋势+消费升级，双栏对比
3. **节假日柱状图**：五大假期六年度全省峰值
4. **桑基图+环形饼图**：客源流向+区域占比
5. **双轴复合图**：线上点评数+好评率趋势
6. **雷达图+恢复指数**：交通方式构成+复苏曲线

## 五、核心发现

- **V型复苏**：2020年三地客流平均降幅59%，2023年全面超越2019
- **消费升级**：敦煌人均消费+72%，兰州+84%
- **客源差异**：敦煌全国性IP（飞机68%），张掖自驾天堂（自驾62%），兰州枢纽均衡
- **淡旺季悬殊**：敦煌淡旺季比8.5:1
- **口碑增长**：三城好评率均超97%

## 六、创新点

1. 多图联动叙事覆盖"热度-客流-消费-客源-口碑"全链路
2. 暗色→暖色主题升级，CSS渐变风景插画替代外部图片
3. 移动端响应式适配
4. 开源数据模拟，可复现验证
