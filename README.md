# 丝路热度数据观察 —— 近五年甘肃各地旅游客流、消费与客源地变迁

## 📁 项目结构

```
D:\数据新闻作品\
├── README.md                       # 项目总览与提交说明
├── 📊 数据\ (5个CSV)                 # 敦煌/张掖/兰州月度数据 + 客源地 + 节假日
├── 🎨 可视化\                        # H5交互作品 + 海报 + Python脚本
│   ├── index.html                  # ★ 主作品 - H5交互数据新闻长页面
│   ├── poster.html                 # ★ 信息海报 - 可打印A3 PDF
│   ├── analysis.py                 # 数据处理分析脚本
│   └── capture_screenshots.py      # 自动截图脚本
├── 📝 文档\                          # 设计说明 + 分析报告
├── 📸 截图\                          # 关键帧截图
└── 📦 作品材料\                      # 最终提交材料
```

## 🚀 快速开始

双击打开 `可视化/index.html`（推荐 Chrome/Edge）

## 📊 可视化图表（8类）

| 图表 | 内容 |
|------|------|
| 热力图 | 敦煌/张掖/兰州月度客流（可切换城市）|
| 折线图 | 2024年三城客流月度趋势 |
| 柱状图 | 2019-2024人均消费年度对比 |
| 节假日柱状图 | 五大假期六年度全省峰值 |
| 桑基图 | 客源流向图 |
| 环形饼图 | 客源区域占比 |
| 双轴图 | 点评数+好评率趋势 |
| 雷达图+恢复指数 | 交通方式+复苏曲线 |

## 📋 提交清单

按 `学号姓名作品名称 材料名称` 格式命名所有文件，不压缩逐个上传：

- 作品PDF（index.html → Ctrl+P 打印为PDF）
- 海报（poster.html → Ctrl+P A3纵向打印为PDF）
- 设计说明 Word + PDF
- 作品关键帧截图（6张板块截图）
- 入围通知/获奖证书/传播数据（如有）

## 🎨 设计

- 暖色沙漠风主题（#f5f0e8 米白底 + #b8860b 金色）
- CSS 渐变风景插画
- 响应式布局（PC+移动端）
- 滚动渐入动画
- ECharts 5.5 数据可视化

数据来源：甘肃省文旅厅、各市州统计局、携程/美团/马蜂窝公开数据（模拟）
